# mezunuyet-Sagl-kl-Sehir

PROJE: Bu siteyi sadece sen değil, bütün mahallenin kullandığını hayal et. Veriler anonim olarak bir havuzda toplanır. Site, belediyeye şu raporu gönderir: "Bu mahallede elektrik tüketimi çok yüksek, buraya acilen güneş paneli teşviği verilmeli." Yani bireysel veriyi, toplumsal bir karar mekanizmasına dönüştürür.
